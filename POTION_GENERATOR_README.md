# 🧪 Генератор зелий DnD

Полнофункциональная система генерации зелий для Dungeons & Dragons 5e, использующая официальную D&D API.

## ✨ Возможности

- **Генерация зелий**: Создание случайных зелий по заданным параметрам
- **Фильтрация по типу**: Восстановление, Усиление, Защита, Иллюзия, Трансмутация, Некромантия, Прорицание, Эвокация, Универсальное
- **Фильтрация по редкости**: Common, Uncommon, Rare, Very Rare, Legendary
- **Реальные данные**: Использует официальную D&D 5e API (dnd5eapi.co)
- **Кеширование**: Автоматическое кеширование для улучшения производительности
- **Адаптивный дизайн**: Красивый и современный интерфейс

## 🚀 Быстрый старт

### 1. Установка

1. Убедитесь, что у вас установлен PHP 7.4+
2. Скопируйте файлы в корневую директорию вашего веб-сервера
3. Убедитесь, что директория `logs/cache/` доступна для записи

### 2. Использование

Откройте `potion-generator.html` в браузере и:

1. Выберите количество зелий (1-20)
2. Выберите тип зелий (или "Все типы")
3. Выберите редкость (или "Все редкости")
4. Нажмите "Сгенерировать зелья"

### 3. Тестирование API

Используйте `test-potion-api.html` для тестирования различных сценариев:

- Случайные зелья
- Фильтрация по типу
- Фильтрация по редкости
- Комбинированные фильтры

### API Endpoints

#### POST /api/generate-potions.php

**Параметры:**
- `count` (int): Количество зелий (1-20)
- `type` (string, опционально): Тип зелья
- `rarity` (string, опционально): Редкость зелья

**Пример запроса:**
```javascript
const formData = new FormData();
formData.append('count', 3);
formData.append('type', 'Восстановление');
formData.append('rarity', 'Rare');

const response = await fetch('/api/generate-potions.php', {
    method: 'POST',
    body: formData
});
```

**Пример ответа:**
```json
{
    "success": true,
    "data": [
{
  "name": "Potion of Healing",
  "rarity": "Common",
  "type": "Восстановление",
            "description": "A character who drinks this magical red liquid regains 2d4 + 2 hit points.",
            "effects": ["Heal", "Restore"],
            "cost": "50 золотых",
            "icon": "🩹"
        }
    ],
    "count": 1,
    "filters": {
        "rarity": "Rare",
        "type": "Восстановление"
    }
}
```

### GET параметры

- `action=rarities` - Список доступных редкостей
- `action=types` - Список доступных типов
- `action=effects` - Список доступных эффектов
- `action=stats` - Статистика зелий
- `action=search` - Поиск зелий
- `action=random` - Случайные зелья

## 🎨 Кастомизация

### Изменение стилей

Все стили находятся в `<style>` секции `potion-generator.html`. Основные CSS классы:

- `.potion-card` - Карточка зелья
- `.potion-header` - Заголовок с иконкой
- `.rarity-*` - Стили для разных редкостей
- `.potion-effects` - Стили для эффектов

### Добавление новых типов зелий

1. Добавьте новый тип в метод `determinePotionType()` в `generate-potions.php`
2. Добавьте соответствующую иконку в метод `getPotionIcon()`
3. Обновите HTML селектор в `potion-generator.html`

## 🐛 Устранение неполадок

### Ошибка "Не удалось получить детальную информацию о зельях"

**Причины:**
- Проблемы с доступом к D&D API
- Неправильная структура данных
- Ошибки в кеше

**Решение:**
1. Проверьте доступность `https://www.dnd5eapi.co/api`
2. Очистите кеш в `logs/cache/`
3. Проверьте логи PHP ошибок

### Медленная работа

**Причины:**
- Отсутствие кеша
- Медленное соединение с D&D API

**Решение:**
1. Увеличьте время кеширования в `generate-potions.php`
2. Используйте CDN для статических файлов
3. Оптимизируйте запросы к API

### Проблемы с фильтрацией

**Причины:**
- Несоответствие названий типов/редкости
- Ошибки в логике фильтрации

**Решение:**
1. Проверьте соответствие значений в HTML и PHP
2. Обновите логику фильтрации в `filterPotionsByCriteria()`
