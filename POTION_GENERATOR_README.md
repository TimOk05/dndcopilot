# 🧪 Генератор зелий DnD

TypeScript/JavaScript компонент для генерации зелий Dungeons & Dragons с использованием официального DnD API.

## ✨ Возможности

- **Выбор количества**: От 1 до 20 зелий
- **Фильтрация по типу**: Исцеляющие, яды, усиливающие, утилитарные
- **Фильтрация по редкости**: Common, Uncommon, Rare, Very Rare, Legendary
- **Реальные данные**: Использует официальное DnD 5e API
- **Fallback система**: При недостатке зелий добавляет случайные
- **Современный UI**: Адаптивный дизайн с анимациями

## 🚀 Быстрый старт

1. **Откройте** `potion-generator.html` в браузере
2. **Выберите параметры**:
   - Количество зелий (1-20)
   - Тип зелий
   - Редкость
3. **Нажмите** "Сгенерировать зелья"

## 🔧 Технические детали

### API Endpoints

- **Базовый URL**: `https://www.dnd5eapi.co/api`
- **Список предметов**: `/magic-items`
- **Детали предмета**: `/magic-items/{index}`

### Фильтрация

#### По типу
- **Healing**: `healing`, `cure`, `heal`
- **Poison**: `poison`, `damage`, `harm`
- **Buff**: `strength`, `power`, `enhance`, `boost`, `advantage`
- **Utility**: `utility`, `tool`, `use`

#### По редкости
- Common (Обычные)
- Uncommon (Необычные)
- Rare (Редкие)
- Very Rare (Очень редкие)
- Legendary (Легендарные)

### Структура данных

```typescript
interface Potion {
  index: string;
  name: string;
  desc: string[];
  rarity?: {
    name: string;
  };
  cost?: {
    quantity: number;
    unit: string;
  };
}
```

## 🎨 Кастомизация

### Изменение стилей
Все стили находятся в `<style>` блоке HTML файла. Основные CSS переменные:

```css
/* Цвета редкости */
.rarity-common { color: #28a745; }
.rarity-uncommon { color: #17a2b8; }
.rarity-rare { color: #6f42c1; }
.rarity-very-rare { color: #fd7e14; }
.rarity-legendary { color: #dc3545; }
```

### Добавление новых типов
В методе `matchesFilters()` добавьте новые case:

```typescript
case 'new-type':
    return name.includes('keyword') || 
           description.includes('keyword');
```

## 🐛 Обработка ошибок

- **API недоступен**: Показывает понятное сообщение об ошибке
- **Нет результатов**: Предлагает рандомные зелья
- **Ошибки сети**: Логирует в консоль и показывает пользователю

## 📱 Адаптивность

- **Desktop**: Полноценный интерфейс с grid-разметкой
- **Mobile**: Вертикальное расположение элементов
- **Breakpoint**: 768px

## 🔄 Fallback система

Если по указанным критериям найдено меньше зелий, чем запрошено:

1. Фильтрует по критериям
2. Добавляет случайные зелья до нужного количества
3. Перемешивает результат

## 🚨 Ограничения

- **Максимум зелий**: 20 (для производительности)
- **API лимиты**: DnD API может иметь ограничения на запросы
- **Фильтрация**: Основана на ключевых словах в названии и описании
