# 🧪 Улучшенный генератор зелий D&D

## 📋 Описание

Улучшенный генератор зелий для D&D, интегрированный в основной сайт DnD Copilot. Использует официальную D&D 5e API для получения реальных данных о зельях.

## ✨ Основные возможности

### 🎯 Генерация зелий
- **Количество**: от 1 до 20 зелий за раз
- **Фильтрация по редкости**: Common, Uncommon, Rare, Very Rare, Legendary
- **Фильтрация по типу**: Восстановление, Усиление, Защита, Иллюзия, Трансмутация, Некромантия, Прорицание, Эвокация
- **Автоматический fallback**: если нет точных совпадений, возвращает случайные зелья

### 🔍 API Endpoints
- `GET /api/generate-potions.php?action=random&count=3&rarity=rare&type=Восстановление`
- `GET /api/generate-potions.php?action=stats` - статистика зелий
- `GET /api/generate-potions.php?action=search&query=healing` - поиск зелий

### 💾 Сохранение
- Автоматическое сохранение зелий в заметки
- Возможность редактирования и удаления заметок
- Интеграция с основной системой заметок сайта

## 🚀 Использование

### 1. Через основной сайт
1. Откройте главную страницу DnD Copilot
2. Нажмите кнопку "🧪 Зелья" в панели быстрых действий
3. Выберите параметры генерации
4. Нажмите "Создать зелья"
5. Сохраните понравившиеся зелья в заметки

### 2. Прямое обращение к API
```bash
# Генерация 3 редких зелий восстановления
curl "api/generate-potions.php?action=random&count=3&rarity=rare&type=Восстановление"

# Получение статистики
curl "api/generate-potions.php?action=stats"

# Поиск зелий
curl "api/generate-potions.php?action=search&query=healing"
```

## 🔧 Технические детали

### Структура данных зелья
```typescript
type Potion = {
  name: string;           // Название зелья
  rarity: string;         // Редкость (Common, Uncommon, Rare, Very Rare, Legendary)
  type: string;           // Тип (Восстановление, Усиление, Защита, и т.д.)
  description: string;    // Описание эффекта
  effects: string[];      // Список эффектов
  value: string;          // Базовая стоимость
  cost: string;           // Точная стоимость из API
  weight: string;         // Вес
  icon: string;           // Эмодзи иконка
  color: string;          // Цвет редкости
  properties: string[];   // Свойства зелья
  equipment_category: string; // Категория снаряжения
}
```

### Фильтрация
- **Редкость**: точное соответствие с API
- **Тип**: определяется по названию и описанию
- **Fallback**: автоматический выбор случайных зелий при отсутствии совпадений

### Кеширование
- Кеш зелий на 1 час для ускорения работы
- Автоматическое обновление при истечении срока
- Сохранение в `logs/cache/potions_cache.json`

## 🐛 Устранение неполадок

### Частые проблемы
1. **API недоступен**: проверьте подключение к интернету
2. **Нет результатов**: попробуйте изменить фильтры
3. **Ошибки кеша**: удалите файл `logs/cache/potions_cache.json`

### Логи
- Ошибки API записываются в консоль браузера
- Проверьте Network tab в DevTools для диагностики
